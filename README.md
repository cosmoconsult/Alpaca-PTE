# COSMO Alpaca AL-Go Per Tenant Extension Template

This repository is a custom template for the [AL-Go-PTE](https://github.com/microsoft/AL-Go-PTE) template and is designed to be used with [COSMO Alpaca](https://cosmoconsult.com/cosmo-alpaca).

Please go to https://aka.ms/AL-Go and [COSMO Docs](https://docs.cosmoconsult.com/en-us/cloud-service/alpaca) to learn more.

> [!TIP]
> Use template [Alpaca-PTE-Template](https://github.com/cosmoconsult/Alpaca-PTE-Template) to create new repositories using this custom template *(recommended)*
